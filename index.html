🔧 Cách lấy Google Client ID

  1. Vào Google Cloud Console → Tạo Project mới (VD: "XMoni")
  2. Vào APIs & Services → Library → Tìm và bật Google Drive API
  3. Vào APIs & Services → OAuth consent screen → Chọn External → Điền tên app, email → Save
  4. Ở tab Scopes → Add scope: drive.file
  5. Ở tab Test users → Thêm email Google của bạn
  6. Vào Credentials → Create Credentials → OAuth client ID
    • Type: Web application
    • Authorized origins: https://<username>.github.io
    • Redirect URIs: https://<username>.github.io/xmoni/
  7. Copy Client ID → paste vào file js/auth.js dòng CLIENT_ID
